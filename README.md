# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of used cars using Machine Learning models like Decision Tree and Random Forest.

The goal is to understand how different features such as year, fuel type, kilometers driven, and engine capacity affect car prices.

---

## 📊 Dataset
- Dataset contains details of cars:
  - Name
  - Year
  - Selling Price
  - KM Driven
  - Fuel Type
  - Transmission
  - Owner Type
  - Mileage
  - Engine
  - Max Power
  - Seats

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 🔄 Workflow
1. Data Collection (CSV file)
2. Data Cleaning (handling missing values)
3. Feature Selection
4. Encoding categorical variables
5. Train-Test Split
6. Model Training (Decision Tree / Random Forest)
7. Prediction
8. Visualization (Actual vs Predicted)

---

## 🤖 Model Used
- Decision Tree Regressor
- Random Forest Regressor (for better accuracy)

---

## 📈 Output
- Predicted car prices
- Scatter plot comparing:
  - Actual Price
  - Predicted Price

---

## 📉 Sample Result
The model shows a strong correlation between actual and predicted values, indicating good performance.
